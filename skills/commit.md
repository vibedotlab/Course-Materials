# Commit

Run lint, fix issues. Run tests — stop and report if they fail.
Check the diff for: debug logs, hardcoded values, TODOs that should be resolved.
Write commit: imperative verb + what changed (72 chars max).
Format: "feat: add email validation" / "fix: resolve login timeout" / "chore: update deps"
Stage and commit. Report what was committed and which tests passed.
Never commit broken code.
